**1. Microsoft Agent Framework 1.0 — унификация AutoGen + Semantic Kernel с поддержкой A2A/MCP**
- Источник: GitHub — Microsoft
- Краткое описание: Microsoft выпустила финальную версию своего фреймворка, объединяющего абстракции AutoGen с enterprise-возможностями Semantic Kernel. Поддержка протоколов A2A (Agent-to-Agent) и MCP (Model Context Protocol) делает фреймворк первым production-ready решением с нативной межплатформенной совместимостью агентов. Фреймворк поддерживает Python и .NET, имеет встроенные observability, security, approvals и долговечное хранение состояния.
- Применение для AstroFinSentinelV5: Активная поддержка A2A/MCP позволяет интегрировать AstroFinSentinelV5 с внешними агентами и инструментами через стандартизированные протоколы — это упрощает построение гибридных архитектур.

**2. HERA — иерархический фреймворк для Multi-Agent RAG с эволюционирующими промптами**
- Источник: arXiv (2604.00901)
- Краткое описание: Новый фреймворк HERA динамически эволюционирует как топологию оркестрации агентов, так и их role-specific промпты на основе накопленного опыта. Показал улучшение на 38.69% на 6 бенчмарках знаниеемких задач и проявляет emergent self-organization.
- Применение для AstroFinSentinelV5: Механизм эволюции промптов агентов на основе реального опыта — это именно то, чего не хватает AstroFinSentinelV5. Интеграция позволила бы агентам автоматически адаптировать стратегии при работе с финансовыми данными.

**3. Orloj — Open-source оркестрация multi-agent AI как Infrastructure as Code (Go, YAML)**
- Источник: GitHub — OrlojHQ/orloj
- Краткое описание: Orloj позиционируется как платформа для production-оркестрации multi-agent систем, построенная на принципах IaC. Агенты, инструменты и политики описываются в YAML, поддерживаются сложные DAG-пайплайны, иерархические графы и swarm-loop топологии.
- Применение для AstroFinSentinelV5: YAML-декларативный подход Orloj отлично подходит для определения финансовых агентов — можно декларативно описывать DAG зависимостей между агентами анализа рынка, риск-менеджмента и портфельной оптимизации.
