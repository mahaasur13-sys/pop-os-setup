## 1. Что нам понадобится

| Что | Сколько | Зачем |
|-----|---------|-------|
| Флешка USB | ≥ 8 ГБ | Загрузочный носитель |
| ISO файл | `pop-os_24.04_amd64_nvidia_23.iso` | Образ системы |
| Компьютер с NVIDIA GPU | RTX/GTX серия | GPU ускорение |
| Стабильный интернет | ≥ 5 ГБ трафика | Установка пакетов |
| Резервная копия данных | На всякий случай | Безопасность |

### Проверка совместимости

Откройте терминал на текущей системе (Windows/Mac/Linux) и проверьте:

```bash
# Проверить, что у вас NVIDIA видеокарта (работает в Windows)
nvidia-smi
```

Должно показать модель GPU и версию драйвера. Если видите таблицу — у вас NVIDIA, продолжаем.

## Шаг 1 — Скачать ISO и создать загрузочную флешку

### 1.1 Скачать ISO

> ⚠️ **Файл уже у вас:** `pop-os_24.04_amd64_nvidia_23.iso`  
> Если нет — скачайте с https://pop.system76.com

### 1.2 Скачать и установить Rufus (только Windows)

1. Скачайте **Rufus** с https://rufus.ie
2. Установите (обычная установка, не портативная версия)

### 1.3 Записать ISO на флешку

> ⚠️ **Все данные на флешке будут удалены!**

1. Вставьте флешку ≥ 8 ГБ
2. Запустите **Rufus**
3. Убедитесь, что в поле **Устройство** выбрана ваша флешка
4. Нажмите кнопку **ВЫБОР** → выберите `pop-os_24.04_amd64_nvidia_23.iso`
5. Схема раздела: **GPT**
6. Целевая система: **UEFI (non-CSM)**
7. Файловая система: **FAT32** (по умолчанию)
8. Нажмите **СТАРТ**
9. Подтвердите предупреждение → **ОК**
10. Дождитесь завершения (5–15 минут)

### Проверка (Linux/Mac)

```bash
# Проверить, что запись прошла успешно
lsblk
# Флешка должна появиться как /dev/sdX с меткой POP-OS
```

### Проверка (Windows)

Откройте Проводник → флешка должна называться **POP-OS** и иметь размер ≈ 4 ГБ.

## Шаг 2 — Загрузиться с USB и установить Pop!_OS

### 2.1 Настроить загрузку с USB

1. **Выключите** компьютер
2. **Включите** и сразу нажимайте:
   - **F2**, **F12**, **Esc** или **Del** — в зависимости от материнской платы
3. В BIOS/UEFI найдите:
   - **Boot Order** или **Загрузка**
   - Переместите **USB** на первое место
4. **Сохраните** настройки и выйдите (F10 → Yes)

### 2.2 Установка Pop!_OS

После загрузки с флешки появится меню:

```
┌─────────────────────────────────────────────┐
│  Pop!_OS 24.04 LTS (NVIDIA)                 │
│                                             │
│  [1] Try or Install Pop!_OS                 │
│  [2] Troubleshooting (Safe Mode)            │
│  [3] Check Disk for Errors                  │
│                                             │
│         Press [1] to continue               │
└─────────────────────────────────────────────┘
```

**Нажмите Enter на [1] — "Try or Install"**

### 2.3 Разметка диска (для новичков — автоматическая)

> ⚠️ **Если у вас только один диск — выберите "Erase and Install"**  
> ⚠️ **Если хотите сохранить Windows — выберите "Alongside"**

| Опция | Что делает |
|-------|------------|
| **Erase and Install** | Удалить всё → установить только Pop!_OS |
| **Alongside Windows** | Сохранить Windows, добавить Pop!_OS рядом |
| **Custom (Advanced)** | Вручную разделить диск (для опытных) |
| **Encrypt** | Шифрование всего диска (пароль при загрузке) |
| **LVM** | Менеджер логических томов (для опытных) |

**Рекомендация для новичка:**

```
✅ Erase and Install (выберите эту, если не знаете что выбрать)
✅ Encrypt the installation (для безопасности личных данных)
```

### 2.4 Выбор пользователя

| Поле | Пример | Что вводить |
|------|--------|-------------|
| **Your name** | Иван | Ваше имя |
| **Your computer's name** | pop-os | Имя компьютера в сети |
| **Pick a username** | ivan | Латиницей, без пробелов |
| **Choose a password** | ••••••• | Минимум 8 символов |
| **Confirm password** | ••••••• | Повторите пароль |

> 📝 Запишите пароль! Без него вы не войдёте в систему.

### 2.5 Завершение установки

1. Установка займёт **10–20 минут**
2. После завершения система предложит **Перезагрузить**
3. **Извлеките флешку** до перезагрузки
4. Нажмите **Enter** на "Reboot Now"

## Шаг 3 — Первая загрузка и настройка сети

### 3.1 Первый вход в систему

После загрузки появится экран входа. Введите **username** и **пароль**, затем нажмите **Sign In**.

### 3.2 Подключение к интернету

> ⚠️ **Это обязательно** — без интернета скрипт не сможет установить пакеты

В правом верхнем углу нажмите на иконку сети и выберите:
- **Wi-Fi** — выберите сеть из списка → введите пароль
- **Ethernet** — подключите кабель (работает автоматически)

**Проверка интернета:**

```bash
# Откройте терминал (Ctrl + Alt + T)
ping -c 3 pop.os
```

Если видите `3 packets transmitted, 3 received` — интернет работает.

### 3.3 Обновление системы (обязательно!)

```bash
sudo apt update && sudo apt upgrade -y
```

Это займёт 5–15 минут. **Не выключайте компьютер.**

## Шаг 5 — Запустить установку (любой профиль)

### 5.1 Сделать скрипт исполняемым

```bash
chmod +x pop-os-setup.sh
```

### 5.2 Запустить установку

> ⚠️ **Минимальная команда (профиль `workstation`):**

```bash
./pop-os-setup.sh
```

> 🚀 **Для профиля `full` (все 26 стадий):**

```bash
./pop-os-setup.sh --profile full
```

> 💻 **Для профиля `ai-dev` (ML/CUDA):**

```bash
./pop-os-setup.sh --profile ai-dev
```

> 🔧 **Для установки конкретных стадий (например, 3, 5, 7):**

```bash
./pop-os-setup.sh --stages 3,5,7
```

### 5.3 Что происходит во время установки


## Если что-то пошло не так

### Чёрный экран после установки NVIDIA

1. Загрузитесь в **Recovery Mode** (из меню GRUB)
2. Выберите **"root — Drop to root shell prompt"**
3. Выполните:
   ```bash
   prime-select nvidia
   reboot
   ```

### Нет интернета

```bash
ip link show
sudo systemctl restart NetworkManager
```

### Скрипт завис

```bash
# Ctrl+C — прервать
# Запустить конкретную стадию:
./pop-os-setup.sh --stages 3  # только NVIDIA
```

### Docker не запускается

```bash
sudo systemctl enable --now docker
sudo usermod -aG docker $USER
# Перелогиньтесь в систему
```
